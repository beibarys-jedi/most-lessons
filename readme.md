## Basic git commands

```git
git init
git add .
git add file-name
git commit -m "type your message"
git push
```

## Take changes from remote to local

1. git pull
2. git fetch
3. git remote update
4. git clone

git pull = git fetch + git merge

## Branch naming commands

1. git checkout -b new-branch-name
2. git checkout branch-name
3. git switch
4. git merge

git pull = git fetch + git merge
